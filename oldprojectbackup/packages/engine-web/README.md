## Core Components

### WebGameRuntime

Complete game engine that runs in browser:

- Full ECS (Entity Component System)
- WebGL rendering with Three.js
- 60+ FPS performance
- Physics simulation
- Component-based architecture
- Scene management
- Camera system
- Lighting system

### WebAssetLoader

Load and manage all assets in browser:

- 3D models (GLTF, OBJ, FBX)
- Textures (PNG, JPG, WebP)
- Audio (MP3, OGG, WAV)
- Scripts (JS, TS)
- Progressive loading
- IndexedDB caching
- Offline support
- CDN optimization

### WebBuildSystem

Build games entirely client-side:

- TypeScript compilation (in browser!)
- Asset bundling
- Code minification
- PWA manifest generation
- Service Worker creation
- One-click deployment
- Downloadable ZIP export
